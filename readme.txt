## Description

Posts Widget Pro is a feature-rich WordPress widget that allows you to display your recent posts with advanced customization options. Whether you want to showcase your latest content, highlight specific categories, or display posts in a custom order, this widget has got you covered.

### Features:
- Display recent posts from your blog
- Advanced sorting options (date, alphabetical, comment count, random)
- Filter posts by categories and tags
- Customizable number of posts to display
- Option to show/hide post date, author, and categories
- Thumbnail support with customizable size
- Responsive design for all devices
- Exclude current post option
- Clean and minimal design
- Easy to set up and use

## Installation

1. Upload the `posts-widget-pro` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Appearance > Widgets** and add the "Posts Widget Pro" widget to your desired widget area

## Frequently Asked Questions

### Can I display posts from specific categories only?

Yes, you can select one or multiple categories in the widget settings to display posts from those categories only.

### Can I change the order of the displayed posts?

You can sort posts by date, alphabetically, by comment count, or even randomly. You can also choose between ascending and descending order.

### Is it possible to show post thumbnails?

Yes, you can enable post thumbnails in the widget settings. You can also choose the thumbnail size (thumbnail, medium, or large).

### Can I exclude the current post from the widget when viewing a single post?

Yes, the widget provides an option to exclude the current post from appearing.
